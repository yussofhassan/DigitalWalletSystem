## Features

- User Registration
- User Login
- Password Hashing (SHA256)
- Add Money
- Send Money
- Change Password
- Transaction History
- Input Validation
- SQL Transactions (Commit / Rollback)

## Database Setup

1. Open SQL Server LocalDB.
2. Run `database.sql`.
3. Start the application.
