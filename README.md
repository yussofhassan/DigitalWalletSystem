# Digital Wallet System

A desktop wallet application built using C#, WinForms and SQL Server.

## Project Overview

This project implements a simple Digital Wallet System with the following features:

- User Registration
- User Login
- Password Hashing (SHA256)
- Add Money
- Send Money
- Change Password
- Transaction History
- Input Validation
- SQL Transactions
