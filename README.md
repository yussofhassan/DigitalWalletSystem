# Digital Wallet System

A Digital Wallet System built using:

- C#
- WinForms
- SQL Server

## Features

- User Registration
- Login System
- Add Money
- Send Money
- Transaction History

## Technologies Used

- .NET WinForms
- SQL Server
- ADO.NET

## How to Run

1. Open SQL Server
2. Run database.sql
3. Open the solution in Visual Studio
4. Run the project
